 OIBSIP-WebDev-L1-Task1-LandingPage
StudyAI is a modern and responsive landing page designed for an AI-powered learning platform. The website introduces the platform, highlights its key features, shares user testimonials, and provides contact information through a clean and engaging interface.

This project was developed as part of the Oasis Infobyte Web Development & Design Internship (Level 1 - Task 1)using HTML5 and CSS3.

Features

1) Sticky navigation bar with smooth scrolling
2) Modern hero section with call-to-action buttons
3) Feature cards showcasing StudyAI services 
4)About section introducing the platform
5)Student testimonials with profile images

6)Responsive footer with contact details and social links

7)Fully responsive design for desktop and mobile devices

8)Clean and modern user interface with a consistent color palette

Technologies Used

HTML5
CSS3
Google Fonts 
CSS Flexbox
CSS Grid

 Home

💻 How to Run

1. Download or clone this repository.
2. Open the project folder.
3. Open **index.html** in any modern web browser.

No additional software or dependencies are required.

Project Objective

The goal of this project is to build a visually appealing and responsive landing page that demonstrates fundamental web development skills using HTML and CSS while following modern UI/UX design principles.
